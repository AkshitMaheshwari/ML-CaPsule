# Mars Weather Prediction using XGBoost

## 📌 Project Overview
This project uses NASA Mars Rover weather datasets to analyze and predict atmospheric conditions on Mars using Machine Learning techniques.

## 📂 Dataset
- **Kaggle Dataset:** [Mars Weather Data](https://www.kaggle.com/datasets/imkrkannan/mars-weather-data)

## ✨ Planned Features
- Data Cleaning & EDA
- Weather Trend Visualization
- Temperature & Pressure Prediction
- ML Model Comparison
- Interactive Dashboard
- Proper Documentation & Contributor Guidelines

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
- XGBoost
- Jupyter Notebook
