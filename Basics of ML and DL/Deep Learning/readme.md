# Deep Learning

## Overview
This folder contains notebooks covering the fundamentals and practical implementation of **Deep Learning**. Deep learning is a subset of machine learning that uses **Artificial Neural Networks (ANNs)** with multiple layers to automatically learn complex patterns from data. These models are widely used in computer vision, natural language processing, speech recognition, and reinforcement learning.

The notebooks in this folder introduce different deep learning architectures and demonstrate their applications using real-world datasets and examples.

## Contents

### 1. Feedforward Neural Network (FNN)
Learn the fundamentals of Feedforward Neural Networks for both **classification** and **regression** tasks. The notebook covers neural network architecture, activation functions, model training, evaluation, and prediction.

### 2. Convolutional Neural Network (CNN)
Build and train a **Convolutional Neural Network (CNN)** for image classification using the **CIFAR-10** dataset. Learn about convolutional layers, pooling layers, feature extraction, and image classification.

### 3. Recurrent Neural Network (RNN)
Implement a **Recurrent Neural Network (RNN)** for sentiment analysis using the **IMDb Movie Reviews** dataset. Understand sequence modeling, word embeddings, and text classification.

### 4. Reinforcement Learning
Explore the basics of **Reinforcement Learning** through **Q-Learning** and **SARSA** algorithms. Learn how an agent interacts with an environment using rewards and penalties to discover the optimal policy.

## Learning Objectives
- Understand the fundamentals of deep learning.
- Learn the architecture of different neural networks.
- Build models for image, text, and tabular data.
- Explore reinforcement learning concepts.
- Evaluate and compare deep learning models.
- Apply deep learning techniques to real-world problems.
