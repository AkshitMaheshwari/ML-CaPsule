# Feedforward Neural Network (FNN) - Classifier and Regressor

## Overview
This notebook introduces **Feedforward Neural Networks (FNNs)** and demonstrates how they can be used for both **classification** and **regression** tasks. An FNN is one of the simplest types of artificial neural networks, where information flows in a single direction—from the input layer to the output layer—without any feedback connections.
